# TrajectoryPredictor

Projeto que visa criar predições de trajetórias de um objeto em um determinado ambiente bidimensional. Esse projeto foi feito para o curso de Machine Learning para Equações Diferenciais do PICME.

Para usar o simulador:
- Crie uma pasta no seu dispositivo e baixe os arquivos ou use o comando `git clone https://github.com/VitinDenoyr/TrajectoryPredictor.git`
- Instale o **Python** e as bibliotecas **PyTorch**, **MatPlotLib** e **NumPy** com os comandos `python -m pip install numpy`, `python -m pip install matplotlib` e `python -m pip install torch`. 
- Dentro da pasta criada, inicie o simulador **gravitational-simulation.py** usando o comando `python gravitational-simulation.py`.
- Have Fun :)
